### Deco and gramma
- Kích cỡ chữ: #, ##, ### có gạch; 4, 5, 6 không gạch
- Tab+'-' dưới '-':  Gạch ngay dưới
- Tab+'-' dưới '#'...:  Code style
- '-' '+' '*': Gạch đầu dòng
- [tên]\(link): Link
- \ + dấu dặc biệt: Gỡ bỏ tác dụng
- ![]\(image.img): Hình ảnh
- [ ]: Dùng để vẽ checkbox unchecked
- [x]: Dùng để vẽ checkbox checked
