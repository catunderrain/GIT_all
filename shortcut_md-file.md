### Deco and gramma
- 1\. 2. 3.:  Chỉ mục
- Kích cỡ chữ: #, ##, ### có gạch; 4, 5, 6 không gạch
- Tab+'-' dưới '-':  Gạch ngay dưới
- Tab+'-' dưới '#'...:  Code style
- '-' '+' '*': Gạch đầu dòng
- [tên]\(link): Link
- \ + dấu dặc biệt: Gỡ bỏ tác dụng
- ![]\(image.img): Hình ảnh
- \*text* hoặc \_: In nghiêng
- \*\*text** hoặc \__: In đậm
- \*\*\*text***: Đậm + nghiêng
- \~~text~~: Gạch bỏ
- \--- = \*** = \___ = #: Tạo đường kẻ
- \` hoặc \```: Code
- \<sub>text\</sub>: Subscript
- \<sup>text\</sup>: Superscript
- \>quote: Quote
- \`#ff00ff`: Color
- [x]: #sd
